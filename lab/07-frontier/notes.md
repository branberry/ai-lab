# Module 07 — Running Notes

Use one section per exercise session. Agents in Learn mode enforce this structure (see `.cursor/skills/learn/SKILL.md`).

Exercise 05 uses Ship mode (`.cursor/skills/ship/SKILL.md`).

## Session template

```markdown
## YYYY-MM-DD — Exercise NN: <title>

### Mental model (before code)
<your explanation in your own words>

### Predictions (before running)
1. ...
2. ...

### What I learned
...

### Surprises / failures of intuition
...

### Open questions
...

### Mastery check
- [ ] Can derive without notes
- [ ] Can predict behavior under perturbation
- [ ] Ready for concept note
```

Polished concept notes go in `docs/concepts/` after the comprehension gate passes.

Weaknesses worth attacking go in `docs/limitations/LL-NNNN-*.md`. Exercise 05 links one to a `TECH-` and eval.
